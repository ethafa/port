Beck-y

A web-application for correcting the posture
while using the computer



Role Concepet development, Research, Design, Prototyping
Teammates Michela Burzio
Supervisors Metteo Loglio Course Machine Learning for Creatives, workshop applying AI

Beck-y apply artificial intelligent technology to help people correcting the posture while using the laptop. Every day millions of people are having troubles in maintaining a straight up position in front of their laptops causing pain and back problems.

There are already applications and devices that are addressing the topic, the difference from those is the ease in the use of Beck-y. Beck-y doesn't need any additional device can run on any computer at any time.

Interface and Interaction modality
Beck-y is a web-application which combines the webcam of the computer with the artificial intelligence technology to track the body position in front of the screen and thanks to a previous calibration It is able to verify If the user is in the correct position. At the beginning It is asked to the user to train the application. The user must register two poses the wrong and the right one. On the side of the page, in the instruction panel, It’s explained that in order to have a valid tracking of the two positions it’s necessary to take at

least thirty takes and moving a little bit while doing that. As a final step the user have to press “save”, by doing that a copy of the calibration will be saved for future uses and the page for the use of the Beck-y tracking will be open. Moreover Beck-y has two different kind of output to prevent a wrong posture: the first one is an sound notification as soon as the user is out of position, secondly If the user maintain an incorrect posture for more than 15 seconds the screen will turn black until the user will get back into the posture setted as the correct one.


User experience
One of the strong aspect of Beck-y is the ease in use of the web-platform. From the calibration of the position to the final output the user is followed step by step. The user has the freedom to set exactly which behave he wants to correct and how he wants to do that. Once the calibration is complete, the web application works autonomously and is activated only when necessary.


Research and development context
More and more people spend several hours working in front of the computer and many of them have a postural problem. Beck-y wants to solve this problem easily without the need for additional devices. This is a possible goal thanks to the application of the latest technologies such as artificial intelligence. One of the possible development is to transform Beck-y from a web-application to an extension of the browser.
