txt.Lab

Is an interactive platform for team members or
individuals to work collaboratively from different locations within the same workspace.



Role Concepet development,
Visual Design, Coding
Teammates Christina Rieder
Course Programming Interactive Objects Supervisors Andreas Gysin

At the start, the team is given a set of textual elements with various fonts and colors. Upon entering the web platform the user is given a random set of letters, each having a different typeface and colour inspired by the designer Bruno Monguzzi. Teams or individuals can work remotely within the same workspace by dragging and dropping the elements provided to create new designs collaboratively.

If a user changes the position of a letter, the others can see the change and react accordingly. If the user wants to save their workspace and also a screenshot of their work, they can click “S” to save what they’ve created onto their desktop. This tool is optimal for designing logos and posters or just to experiment with text elements and colors.

The iteration
txt.Lab is an interactive platform for team members or individuals to collaborate using textual design elements. From our experiences of working together in teams, we enjoy tools that allow for real time collaboration

in order for flexibility and expression of each individual team member within the whole. We also have respect for confinements during the creative process, in order to narrow the creative direction and generate results within a shorter amount of time.


Interaction modalities
Upon entering the web platform the user is given a random set of letters, each having a different typeface and colour inspired by the designer Bruno Monguzzi. Teams or individuals can work remotely within the same workspace by dragging and dropping the elements provided to create new designs collaboratively. If a user changes the position of a letter, the others can see the change and react accordingly. If the user wants to save their workspace and also


a screenshot of their work, they can click “S” to save what they’ve created onto their desktop. By clicking “K” the user can also save their layout in the form of a text file, this allows them to store and exchange a lighter file format. This experience is collaborative and playful, and it provides constraints as a tool for creativity. Our working prototype is crafted using processing and can be viewed through a browser by sharing a link to the members of the team.
